# AI Product Content Generator

## Overview
This project builds an automated pipeline that:
- Fetches product data from an API
- Uses an LLM to generate SEO descriptions
- Stores results locally and in MySQL

## Tech Stack
- Python
- Requests (API calls)
- OpenAI API (LLM)
- SQLite
- JSON

## How to Run
1. Add API key in .env
2. Run:
   python app/main.py
